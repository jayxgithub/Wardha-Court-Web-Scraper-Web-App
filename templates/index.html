How to Use This System

  • Case Type: Select the appropriate case type from the dropdown menu
  • Case Number: Enter the exact case number as registered in court
  • Filing Year: Enter the year when the case was filed in court
  • Example: Civil Suit 123/2024 (Type: Civil Suit, Number: 123, Year: 2024)
  • Format: Case numbers can include numbers, slashes (/) and dashes (-)
